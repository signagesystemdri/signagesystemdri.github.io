Modular Panel Signage System

For the Studio Areas, several Modular Panels are installed close to the access points. This Panels are designed with a grid system that allows adding different contents due to its modular structure.

Modular panel design

Studio Area (3rd, 4rth and 5th Floor).
‍
This design system was created in response to the need of communicating and guiding users to attend ephemeral events such as classes, workshops, presentations and specific events. In this way, this system is customizable to include permanent and temporary information relevant to special events in order to inform and guide users in a simple, fast and effective way.
At the end of every event, the temporary information can be removed without making a significant effort.
‍
This panel has a grid system that allows adding different contents due to its modular structure. Contents like Signs or posters can be used to display both permanent and temporary information. These panels are mounted on the walls near the main access points of each floor and also display the location plan, floor overviews, and destination confirmation. However, these signs are not fixed and can be easily replaced.

Materials
‍
The panels are made of stainless steel with coating methods. These panels are fully perforated following a symetric pattern to allow the installation of external elements mounted over the surface. To install these external elements it is required to be fastened with some special metal buttons. There are two types of panels, both located at access points: a) Full size main panels located at the entrance to each floor of the building with a standard size of 1650 cm x 500cm.  b) Small Size Panels located at the access points of each classroom with a standard size of 320 x 180 cm.


Installation
‍
For this system it is necessary to attach a certain type of steel brackets to fix the signals on the panel. These can be secured by hand and without the need for any external tools. To include the printed materials it is necessary that these materials have a perforation on the edges, however, both printing and manufacturing can be completed within the DRI's facilities.
